# VYRA — Rede Profissional do Futuro (Front-end)

## Resumo do projeto
Aplicação web (SPA) construída com **React + Tailwind CSS** que simula uma rede profissional voltada ao futuro do trabalho. Permite listar perfis simulados (JSON local), filtrar/buscar por cidade/área/tecnologia e abrir modal com detalhes do perfil. Tem ações locais (recomendar, enviar mensagem) usando `localStorage`, design responsivo e modo escuro.

## Usuários e senhas
- Não existem usuários/credenciais embutidas no projeto (autenticação não implementada).
